# 🗡️ Kill Bill Theme CV Template

An interactive Portfolio website inspired by the Kill Bill film aesthetic.It combines cinematic design elements with modern web technologies to create an unforgettable portfolio experience.

## ✨ Features

### 🎬 Kill Bill Aesthetic
- **Iconic Color Scheme**: Yellow and black inspired by Kill Bill Vol. 1 & 2
- **Cinematic UI**: Bold typography using Oswald and Roboto Condensed fonts
- **Interactive Animations**: Parallax effects, custom cursors, and dynamic canvas renderings

### 🎵 Built-in Jukebox
- **Interactive Music Player**: Fixed jukebox widget with playlist controls
- **Curated Soundtrack**: Multi-lingual international music collection
- **Shuffle & Skip**: Full playback controls for seamless audio experience

### 🎨 Visual Effects
- **Custom Katana Cursor**: Physics-based animated sword cursor (non-interactive elements)
- **Parallax Background**: Dynamic canvas with drifting text and blood splatter effects
- **Revenge Overlay**: Cinematic radial gradient overlay for dramatic effect
- **3D Perspective**: Hover effects on CV cards with 3D transforms

### 📱 Responsive Design
- **Modern Layout**: Clean container system with proper spacing
- **Smooth Transitions**: CSS animations and transitions throughout
- **Custom Scrollbar**: Enhanced user experience

## 📁 File Structure

- **index.html** - Main CV template with complete structure and styling
- **jukebox.js** - Interactive music player module with playlist management
- **parallax.js** - Canvas-based background effects and animations
- **track*.mp3** - Audio files for the jukebox playlist

## 🚀 Usage

1. Open `index.html` in a modern web browser
2. Click the "INITIALIZE" button to enter the CV
3. Enjoy the interactive experience with background music and visual effects
4. Use the jukebox (top-right) to control music playback

## 🛠️ Technical Stack

- **HTML5**: Semantic markup and canvas integration
- **CSS3**: Advanced styling with custom properties, gradients, and transforms
- **JavaScript (ES6+)**: Interactive features, event handling, and animations
- **Canvas API**: Dynamic background rendering and effects

## ⚙️ Customization

1. **Colors**: Edit CSS variables in `index.html`:
   - `--bill-yellow`: Primary yellow
   - `--bill-black`: Primary black
   - `--bill-red`: Accent red
   - `--arterial-red`: Secondary red

2. **Content**: Update the CV content directly in `index.html`
3. **Playlist**: Modify the track list in `jukebox.js`
4. **Effects**: Adjust animation parameters in `parallax.js`
